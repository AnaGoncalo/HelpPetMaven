API HelpPet

Denuncia

/rest/denuncia
    GET: retorna uma lista com todas as denúncias cadastradas.
    POST: requere uma denúncia em json.

Animal

/rest/animal
    GET: retorna uma lista em JSON com todas os ANIMAIS, não adotados ou não achados ou não resgatados, cadastrados.
    POST: requere um ANIMAL em JSON para cadastrar.
    PUT: requere um ANIMAL em JSON para atualizar.
    DELETE: requere um ANIMAL em JSON para excluir.
/rest/animal/{idAnimal}
    GET: requere um idAnimal em INT e retorna um ANIMAL em JSON.

Usuario

/rest/usuario
    POST: requere um USUARIO em JSON para cadastrar.
    PUT: requere um USUARIO em JSON para atualizar.
    DELETE: requere um USUARIO em JSON para excluir.
/rest/usuario/{idUsuario}
    GET: requere um idUsuario em INT e retorna um USUARIO em JSON.
/rest/usuario/{idUsuario}/animal
    GET: requere um 'idUsuario' em INT e retorna uma lista em JSON com os ANIMAIS cadastrados para o respectivo usuario
/rest/usuario/{idUsuario}/anuncio
    GET: requere um 'idUsuario' em INT e retorna uma lista em JSON com os ANUNCIOS cadastrados para o respectivo usuario
/rest/usuario/{idUsuario}/estoque

/rest/usuario/{idUsuario}/encontro

/rest/usuario/{idUsuario}/experiencia

Login

/rest/logar
    POST: requere um USUARIO em JSON para verificar login, retorna um USUARIO se o login ocorreu com sucesso, se não retorna NULL.
